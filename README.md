## Getting Start

星搭拓展开发流程，可以参考[这篇文章](https://docs.staringos.com/?path=/docs/%E6%8B%93%E5%B1%95-%E5%BF%AB%E9%80%9F%E5%BC%80%E5%A7%8B--page&f=mtbird-extension-ai-assistants)

```js
yarn
yarn run start
```

在 `mtbird.staringos.com` 编辑器中，点击调试按钮，输入命令行输出的链接，进行调试
